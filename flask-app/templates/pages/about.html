Tentang Proyek

Menjelajahi konsep Matriks dan Machine Learning melalui aplikasi web interaktif.

Website ini dikembangkan sebagai bagian dari mata kuliah Pemrograman 3. Tujuannya adalah membantu mahasiswa memahami konsep Matriks dan penerapannya dalam Machine Learning melalui visualisasi interaktif dan perhitungan otomatis.

Aplikasi ini menggunakan dataset CSV sebagai sumber data untuk operasi matriks, regresi linear, dan analisis pembelajaran mesin berbasis Normal Equation.

Fungsionalitas Utama

  • Import CSV & Operasi Matriks: Mengimpor data dan mengolah dengan metode Transpose & Invers.
  • Algoritma Regresi Linear: Menerapkan Persamaan Normal (XᵀX)⁻¹XᵀY.
  • Visualisasi Data: Menampilkan garis regresi dan hasil prediksi dengan Chart.js.

Teknologi & Prinsip

  • Clean Code: Prinsip SOLID diterapkan dalam arsitektur modul matriks/.
  • Containerization: Proyek siap dijalankan di Docker.
  • Kolaborasi: Menggunakan Git untuk kerja tim dan manajemen versi.
