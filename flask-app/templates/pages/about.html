Project UTS Pemrograman 3

Proyek ini mengintegrasikan konsep **Pemrograman Berorientasi Objek (OOP)**, **Clean Code (SOLID)**, dan **Machine Learning (Regresi Linear)** ke dalam aplikasi web sederhana menggunakan *framework* **Flask**.

Fungsionalitas Inti:

  • **Import CSV & Matrix Operation:** Mengimpor data dari CSV dan mengolahnya menggunakan implementasi Matriks kustom (termasuk Transpose dan Inverse).
  • **Algoritma Regresi Linear:** Menerapkan Persamaan Normal ($W = (X^T X)^{-1} X^T Y$) untuk melatih model.
  • **Visualisasi Data:** Menampilkan hasil prediksi dan garis regresi menggunakan Chart.js.

Teknologi & Prinsip yang Diterapkan:

  • **Clean Code:** Penerapan prinsip **SOLID** (terutama SRP dan OCP) dalam arsitektur modul `matriks/`.
  • **Containerization:** Proyek siap di-*deploy* menggunakan **Docker**.
  • **Kolaborasi:** Menggunakan **Git** untuk kolaborasi tim.
